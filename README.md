# Failide genereerimine Pythoniga

Loodud õppe eesmärgil õpilastele, kiiresti erinevate failide tegemiseks. Eeldab Pythoni olemasolu. Õpitakse käsurealt kloonima.

# Paigaldamine käsureal
1. Käivita terminal (Windows 11 on see PowerShell)
2. Liigu kausta kuhu soovid projekti kloonida. Eeldan et oled "kasutaja" ning soovid kausta "Documents" kloonida
   - ```cd c:\Users\kasutaja\Documents```
3. Klooni projekt kirjutades käsureale
   - ```git clone https://github.com/OkramL/Generate-Files.git```
   - Juhul, kui, soovid teise nimega projekti (praegu on Generate-Files), siis 
    - ```git clone https://github.com/OkramL/Generate-Files.git SINU_VALITUD_NIMI```
4. Liigu projekti kausta
    - ```cd Generate-Files```

# Käivitamine käsurealt

Käivitamiseks pead olema õiges kaustas ning siis anna käsklus
```py main.py``` Täida ekraanil esitatud küsimused. Hoolega jälgi kolmandat ehk viimast küsimust. Ära siin eksi! Soovituslik koht, kuhu failid teha on ```c:\Temp\files```

# Mine õigesse kausta
```cd c:\Temp\files```
